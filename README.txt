Application AndroTaMa

l'exutable est src/main.py à lancer sans aucun argument de ligne de commande

Il vous sera demandé de choisir entre générer une politique ou en charger une.
Il est demandé de ne rentrer que des entiers lors des invités de commande.
Pour le choix de génération ne sélectionner que un seul types de Bases de données

La classe main.py contient les coefficients de l'algorithme initialisé par défaut, modifiable avec un éditeur de texte.
